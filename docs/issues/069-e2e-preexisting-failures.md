Playwright 액셔너빌리티 대기 → 052 타임아웃으로 나타났다.
**제품 동작은 전부 정상**(드릴 = 승인 UX, 그립 밴드 11px = 031 의도된 어포던스, 빈 공간 클릭 해제 =
QA2 #2 수정) → **테스트만 드릴 모델로 정렬**: 세 스펙의 셀 스캔 헬퍼를 "단일클릭(표 마킹) → 500ms
대기(앱 더블클릭 감지 윈도 회피) → 같은 지점 빠른 2클릭(셀 드릴)"로 교체, 그립 인터셉트 행에는
절대좌표 `page.mouse`(액셔너빌리티 미대기)를 사용.
